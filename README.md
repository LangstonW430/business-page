# Freelance Services Page

A freelance web design landing page targeting local service businesses — landscapers, cleaners, contractors, and similar trades. Built to integrate with [langstonwoods.com](https://langstonwoods.com) and convert small business owners into clients.

## Stack

- React 19 + TypeScript
- Vite
- Plain CSS (design system matched to langstonwoods.com)

## Design

Dark-mode adaptation of the langstonwoods.com design system — same typefaces (Playfair Display, DM Mono, DM Sans), same editorial patterns (section labels, sliding-fill buttons, left-border card hovers), translated into a warm dark palette (`#0e0b09` background, `#c4500a` rust accent, `#f5f0e8` cream text).

## Sections

1. Hero — value proposition with live GreenEdge site preview
2. Who It's For — target business types
3. The Problem — why local businesses lose customers online
4. What I Build — services offered
5. Real Work — GreenEdge Landscape & Design ([live](https://mock-landscaper.vercel.app/) · [code](https://github.com/LangstonW430/mock-landscaper))
6. Pricing — Starter ($499) and Growth ($899) tiers
7. Process — 4-step workflow
8. CTA — contact section
